# RG Game - 已實作功能清單 (Implemented Features)

本文件摘要了目前對遊戲所做的重大改裝與功能更新，作為之後開發的參考。

## 1. 視覺與美術資產 (Visuals & Assets)
*   **主角更換**: 使用 `robot.png` 作為玩家角色。
    *   **即時去背技術**: 實作了 Canvas 影像處理邏輯，啟動時自動過濾掉圖片中的純白色背景，解決貼圖白邊問題。
    *   **角色比例**: 調整至 **0.04**，使其符合地圖的大視角比例。
*   **超載模式貼圖**: 當玩家啟動超載模式時，角色外觀切換為 `Robot2.png`（進階機器人形態）。
    *   **放大效果**: 超載狀態下角色放大至 **0.75** 倍（約為正常狀態的 18.75 倍），視覺上更具威壓感。
    *   **同樣去背處理**: Robot2 貼圖也套用了自動去背技術，確保視覺一致性。
*   **中世紀/皇宮背景**: 使用 `BG1.png`（浮空島皇宮）作為無縫拼接地圖 (`tileSprite`)。
    *   **層次感優化**: 對背景套用了調暗與去色處理，強調前景角色。
    *   **無限地圖**: 背景會隨著玩家移動而無縫接續，實現真正的無邊界探索體驗。
*   **木箱子貼圖**: 箱子改用 `chest.png`（木質板車/板箱圖案），並放大至 **1.5x** 以利辨識。

## 2. UI/UX 增強 (Glassmorphism UI)
*   **全域 UI 風格統一**: 所有面板（HP、統計、計時、倍速）均採用半透明深色毛玻璃背景。
*   **經驗值條 (XP Bar)**: 在畫面底端實作了全螢幕寬度的綠色經驗條。
*   **HP 面板**: 左上角精緻的血量顯示條與等級資訊。
*   **計時器**: 頂部中央的懸浮計時面板。
*   **屬性面板**: 右上角顯示即時的攻擊力、攻速與跑速倍率。

## 3. 戰鬥系統 (Combat Systems)
*   **魔王追蹤指北針 (Boss Indicator)**: 當 Boss 在視野外時，螢幕邊緣會出現箭頭標示其方向與距離（公尺）。
*   **箱子自動鎖定**: 所有的武器系統（彈道、近戰、光環）現在都會將箱子視為有效攻擊目標。
*   **武器升級機制**: 整合了傷害、範圍、攻速與穿透力的升級邏輯。

## 4. 操作與控制 (Performance & Controls)
*   **遊戲倍速切換**: 支援 `0.5x`, `1x`, `2x`, `3x` 四種倍速。
    *   **快捷鍵支援**: 可使用鍵盤 `+` (或 `Numpad +`) 加速，`-` (或 `Numpad -`) 減速。
*   **流暢移動**: 角色移動精度優化，確保在高倍速下依然能精準追隨滑鼠。
