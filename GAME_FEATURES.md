# RG Game - 已實作功能清單 (Implemented Features)

本文件摘要了目前對遊戲所做的重大改裝與功能更新，作為之後開發的參考。

## 1. 視覺與美術資產 (Visuals & Assets)
*   **主角系統 (Player System)**:
    *   **基礎貼圖**: 使用 `robot.png` 作為核心角色原型。
    *   **行走動畫**: 導入了 `Main1.png` 至 `Main4.png` 動態序列，呈現流暢的步行感。
    *   **即時去背技術**: 實作了 Canvas 影像處理邏輯，啟動時自動過濾掉 `robot.png` 與走動貼圖中的純白色背景。
    *   **角色比例**: 調整至 **0.04** 以符合地圖視角。
*   **超載模式 (Overload Mode)**:
    *   **進階形態**: 啟動時切換為 `Robot2.png`。
    *   **威壓感強化**: 角色放大至 **0.75** 倍（約為常規狀態的 18.75 倍），並套用自動去背處理。
*   **環境背景 (Environment)**:
    *   **地圖更新**: 使用 `BG4-2.png` 作為無縫拼接地圖 (`tileSprite`)，取代舊版的 BG1。
    *   **視覺優化**: 背景具備無限拼接特性，並對色調進行微調以突顯戰鬥前景。
*   **物件貼圖**:
    *   **木箱**: 改用 `chest.png` 並放大至 **1.5x**，整合進戰鬥鎖定系統。

## 2. 戰鬥系統 (Combat Systems)
*   **魔王追蹤指北針 (Boss Indicator)**: 螢幕邊緣指向 Boss 方向與距離（單位：公尺）。
*   **武器鎖定優化**: 武器系統（包含彈道、近戰與須佐能乎）將木箱視為有效攻擊目標。
*   **升級路徑**: 實作了 1-5 級的武器成長路徑，涵蓋 DPS、範圍與特殊效果。

## 3. 操作與控制 (Performance & Controls)
*   **時空節奏切換**: 支援 `0.5x` 到 `3x` 的遊戲速度切換。
*   **快捷鍵支援**: 支援鍵盤 `+` / `-` 控制速率。
*   **測試與除錯模式 (Debug Mode)**: 
    *   **快速切換**: 遊戲中按下 `Tab` 鍵可啟動除錯測試面板。
    *   **自動暫停**: 開啟除錯面板時，遊戲邏輯與計時器會自動進入暫停狀態，方便開發人員即時調整參數。
    *   **即時調整**: 面板支援一鍵殺死全場敵人、獲取經驗值、調整全武器等級以及切換無敵狀態。

## 🏛️ 高品質遊戲 UI 規範 (Professional UI Design Guidelines)

作為遊戲 UI 設計師，本專案採用的「黑曜石與金箔」程序化風格遵循以下設計手冊：

### 🎨 設計語彙與材質 (Materials & Texture)
*   **黑曜石 (Obsidian)**：主要結構材質。色碼 `#1a1a1a`，具備深色且半啞光的質感。
*   **拋光金箔 (Gold Filigree)**：重點細節裝飾。色碼 `#daa520`，用於邊界飾線與符文連接，強調古老的神祕感。
*   **羊皮紙捲軸 (Aged Parchment)**：資訊承載材質。色碼 `#d2b48c`，帶有歲月感的暖色調，用於屬性展示。

### ✨ 視覺特效與動態 (Visual Effects & Motion)
*   **流體動力學 (Fluid Physics)**：資源球（HP）內的液體需實作三層正弦波 (Sine Wave) 動態混合。
*   **受傷震盪 (Impact Shake)**：UI 必須具備動態物理反饋。受傷時，資源球液體震幅瞬間激增並呈指數衰減，模擬容器受撞擊的物理性。
*   **混合發光 (Magical Glow)**：關鍵元件周圍套用 `SCREEN` 或 `ADD` 混合模式，使金屬與液體產生魔法光澤。

### 📐 佈局與尺寸技術指標 (Dimensions & Grid)
*   **邏輯解析度**：固定 **1280x720 (16:9)**。UI 元件座標固定，確保跨裝置不移位。
*   **間距系統 (8pt Grid)**：所有邊距與元件間距需符合 8 的倍數。
*   **尺寸規範**：
    *   **主要圖示**：48px - 64px。
    *   **屬性文字**：14px - 16px (Berkshire Swash)。
    *   **關鍵數值**：24px - 32px (Cinzel, 帶 3px 描邊)。

### 📊 資訊階層配置 (Information Hierarchy)
1.  **生存核心 (Vitality)**：左上角 Master HUD。整合 HP 核心、Essence (XP) 槽、Vana (能量) 槽。
2.  **戰鬥統計 (Metrics)**：右上角魔導書。顯示 DPS、傷害倍率、速度加成。
3.  **環境資訊 (Meta)**：頂部懸浮計時器、右下角擊殺統計。
