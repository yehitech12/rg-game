# ⚔️ 武器研發與升級需求表 (Weapon Evolution Cards)

本文件紀錄所有武器的基礎參數與「1-5 級」升級路徑。
**DPS 計算公式**: `(傷害 * 發射數量 * (1 + 彈跳次數 * 0.5)) / (攻速 / 1000)`
*註：彈跳次數因並非每次都能跳滿，故以 0.5 倍率折算入理論 DPS。*

## 🟣 魔法：須佐能乎 (Susanoo)
### [基礎參數 LV1]
- **傷害**: 50 (掃擊), 20 (推開)
- **冷卻**: 2500 ms (每次行動間隔，不受遊戲加速影響)
- **範圍**: 250 px (防守範圍)
- **護盾回復**: 20
- **其他效果**: 自動防禦反擊系統，紫色查克拉巨人視覺特效

### [護盾功能詳細說明]
須佐能乎會根據戰況自動選擇三種行動模式：

#### 1. 🛡️ **護盾模式 (Shield)**
**觸發條件**：
- 玩家 HP < 40% **或**
- 玩家護盾值 ≤ 0 

**效果**：
- 立即恢復護盾值（基礎 20，LV4 後 40）
- 播放防禦動畫（SUS_def）
- 冷卻時間：30000ms（30秒）
- 優先級：**最高**（生存優先）

#### 2. 💨 **推開模式 (Push)**
**觸發條件**：
- 敵人距離 < 200px（非常近）

**效果**：
- 造成推開傷害（基礎 20，LV3 後 40）
- 將敵人擊退 300px
- 播放攻擊動畫（SUS_att）
- 冷卻時間：1500ms
- 優先級：**中**（自衛反擊）

#### 3. ⚔️ **掃擊模式 (Sweep)**
**觸發條件**：
- 敵人在攻擊範圍內（基礎 250px，LV3 後 500px）
- 敵人距離 ≥ 130px（不太近）


**效果**：
- 造成掃擊傷害（基礎 50，LV3 後 80）
- 虛空風格紫色斬擊特效（150px 弧形範圍）
- 播放攻擊動畫（SUS_att）
- 冷卻時間：1000ms
- 優先級：**低**（主動攻擊）

### [發展路徑]
- **LV1**: 初始狀態 (推開 20dmg / 掃擊 50dmg / 護盾回復 20) | **DPS: 8.3**
- **LV2**: 冷卻 -500ms (2000ms) 攻擊更頻繁 | **DPS: 8.3**
- **LV3**: 傷害 +30 (推開 40 / 掃擊 80) **+ 範圍 +250 (500px)** | **DPS: 13.3**
- **LV4**: 護盾回復 +20 (40) 生存力大幅提升 | **生存力 UP**
- **LV5 (MAX)**: 冷卻 -500ms (1500ms) 終極守護頻率 | **DPS: 22.2**

### [特殊機制]
- 須佐能乎圖層位於主角**下方**（背後）
- 移動動畫：2 FPS（緩慢呼吸感）
- 半透明效果（alpha 0.5）
- 超載模式下自動隱藏
- 攻擊速度**不受時間加速影響**（使用真實時間計算）
